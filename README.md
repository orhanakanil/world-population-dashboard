🌍 World Population Dashboard

Bu proje, **RestCountries API** kullanarak dünya ülkelerini listeleyen, nüfusa göre sıralayan ve arama yapılabilen dinamik bir web uygulamasıdır.

Özellikler
- **Fetch API & Async/Await:** Gerçek zamanlı veri çekme.
- **Dinamik Filtreleme:** Ülke ismine göre anlık arama.
- **Sıralama:** Nüfusa göre (Artan/Azalan) sıralama seçenekleri.
- **Veri Hesaplama:** `reduce` metodu ile ekrandaki ülkelerin toplam nüfusunu hesaplama.

Kullanılan Teknolojiler
- HTML5 & CSS3
- JavaScript (ES6+, Map, Filter, Sort, Reduce, Destructuring)
